
## 🛠️ Requirements

- Java JDK (version 8 or above)

## 🔧 How to Run

1. **Clone or Download the Repository**

    ```bash
    git clone https://github.com/your-username/NotesApp.git
    cd NotesApp
    ```

2. **Compile the Java File**

    ```bash
    javac NotesApp.java
    ```

3. **Run the Application**

    ```bash
    java NotesApp
    ```

4. **Follow the Menu Options in the Terminal**
